# H2O Swimming Club

Готовый адаптивный одностраничный сайт. Использованы только предоставленные фотографии H2O.

Откройте `index.html` в браузере. Для публикации загрузите все файлы и папку `assets` в корень GitHub-репозитория, затем включите GitHub Pages в Settings → Pages.

Форма заявки пока демонстрационная: для реальной отправки её нужно подключить к WhatsApp, Telegram, почте или сервису форм.
